O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web. Ele define a estrutura do conteúdo de um site e é interpretado pelos navegadores para exibir textos, imagens, vídeos e outros elementos de forma organizada. O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação, ou seja, ele "marca" o conteúdo para que o navegador saiba como exibi-lo.
